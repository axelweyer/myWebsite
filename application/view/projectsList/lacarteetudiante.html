PROJECT



La Carte Etudiante

AUG 2016 - Current



Avez-vous déjà galéré pour trouver un logement étudiant près de votre école ? Ou alors avant les études, un problème pour connaître les écoles dans votre secteur ? Ce projet est pour vous : étudiants et futurs étudiants.

Encore en phase d'idéation, le projet a besoin de votre aide pour finaliser l'objectif final. Le projet est une application web communautaire ; elle est composée d'une Map interactive avec les différentes résidences universitaires et les écoles. Tout le monde peut ajouter un bâtiment sur la carte, et quand celui-ci sera validé, les différentes informations seront affichées (ex : adresse, téléphone, photos etc). Vous pourrez aussi ajouter des avis sur ces lieux fréquentés pour aider les internautes dans leur choix de résidences et écoles.

Dans une seconde partie, un coin pour les vendeurs sera en place. Sur le même principe d'une Map, les ventes seront triées par résidences étudiantes. Les étudiants pourront alors écrire des annonces pour vendre leur meubles par exemple. Bien sûr, l'accès au site sera totalement libre et gratuit !
